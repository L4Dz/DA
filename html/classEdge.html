Detailed Description

template<class T>
class Edge< T >

(EN) Represents a directed weighted edge with flow support between two vertices.

(PT) Representa uma aresta dirigida e pesada com suporte a fluxo entre dois vértices.

Template Parameters
TType of the vertex info. / (PT) Tipo da informação dos vértices.

(EN) Each edge stores its origin and destination vertices, a capacity (weight), a current flow value, a selected flag (for path marking), and an optional reverse edge pointer (used in residual graph construction for flow algorithms).

(PT) Cada aresta armazena os vértices de origem e destino, uma capacidade (weight), um valor de fluxo atual, um flag de seleção (para marcar caminhos), e um ponteiro opcional para a aresta inversa (usado na construção do grafo residual para algoritmos de fluxo).
