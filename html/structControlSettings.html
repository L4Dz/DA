Detailed Description

(EN) Holds runtime control flags and output configuration read from the CSV.

(PT) Contém flags de controlo de execução e configuração de saída lidos do CSV.

(EN) These settings are read from the [Control] section of the CSV input file and determine which operations are performed and where the results are written.

(PT) Estas definições são lidas da secção [Control] do ficheiro CSV de entrada e determinam quais operações são executadas e onde os resultados são escritos.
