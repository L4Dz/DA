Detailed Description

template<class T>
class Vertex< T >

(EN) Represents a node in the graph, carrying adjacency lists and traversal state.

(PT) Representa um nó no grafo, com listas de adjacência e estado de travessia.

Template Parameters
TType of the identifying information stored in the vertex.
T(PT) Tipo da informação identificadora armazenada no vértice.

(EN) Each vertex stores outgoing edges (adj), incoming edges (incoming), and fields used by graph algorithms: visited, processing, dist, path, low, num. Public metadata fields (type, title, primaryDomain, secondaryDomain) are used to model the submission-reviewer assignment problem.

(PT) Cada vértice armazena arestas de saída (adj), arestas de entrada (incoming), e campos usados pelos algoritmos do grafo: visited, processing, dist, path, low, num. Os campos públicos de metadados (type, title, primaryDomain, secondaryDomain) são usados para modelar o problema de atribuição de submissões a revisores.
