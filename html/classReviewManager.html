◆ findRiskyReviewers()

std::vector< int > ReviewManager::findRiskyReviewers ( )
inline

Identifies reviewers whose absence would compromise the minimum review requirements.

Acts as a Risk Analysis (K=1) heuristic. Checks if removing an active reviewer causes any submission's pool of valid candidates to drop below minReviews.

Note
Time Complexity: $O(V \cdot E)$ where V is the number of submissions and E is their adjacency list size.
Returns
std::vector<int> A sorted list of risky reviewer IDs.
