Detailed Description

(EN) Holds algorithm parameters that control how reviewer-submission matching is performed.

(PT) Contém os parâmetros do algoritmo que controlam como é feita a correspondência revisor-submissão.

(EN) These parameters are read from the [Parameters] section of the CSV input file. They determine the minimum number of reviews each submission must receive, the maximum workload per reviewer, and which domain expertise combinations are considered valid matches.

(PT) Estes parâmetros são lidos da secção [Parameters] do ficheiro CSV de entrada. Determinam o número mínimo de revisões que cada submissão deve receber, a carga máxima por revisor, e quais combinações de domínio de especialização são consideradas correspondências válidas.
